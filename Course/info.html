Справка по использованию SuperBMP 2019

Меню File

В меню File находятся команды открытия, закрытия и создания изображения.

Open

Команда Open открывает изображение в формате .bmp версий 3, 4 и 5 и выводит его на экран.

Create

Создает .bmp файл версии 5 размером 1000x1000 пикселей.

Save

Сохраняет открытый или созданный в приложении .bmp файл в выбранной директории.

Меню Functions

Содержит набор команд для работы с .bmp изображением.

Find rectangle

Открывает диалоговое окно, в котором пользователь вводит цвет прямоугольников, которые необходимо найти в изоображении, и обводит их с заданной пользователем цветом и толщиной.

Drawing a circle

Открывает диалоговое окно, в котором пользователь вводит для рисования окружности на изображении координаты центра, радиус, толщину обводки и её цвет, наличие заливки и её цвет.

RGB Filter

Открывает диалоговое окно, в котором пользователь выбирает RGB-канал и его значение, которое необходимо применить ко всему изображению.

Separate image

Открывает диалоговое окно, в котором пользователь вводит количество частей по оси X и Y, на которые необходимо разделить изображение, и цвет и толщину линий-разделителей.

Меню Info

Содержит информацию о файле и приложении.

About File

Выводит информацию о файле, а именно его разрешение в пикселях, размер на диске и глубину цвета в битах на пиксель.

Author

Выводит сведения об авторе.

Help

Открывает справку.
